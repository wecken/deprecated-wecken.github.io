Mercedes-Benz User Experience (MBUX)

2019

I worked at Mercedes-Benz Japan as a UI/UX Expert to improve the usability of brand's in-car infotainment system called MBUX. I was responsible for its GPS navigation features for Japanese market.

Optimizing user interaction, and functionality, user experience of the system is significantly improved.

Privious model was featuring address input method where user had to choose it step by step from prefecture to street number. Mercedes-Benz was thinking this was safer while driving. However, this method took 90 seconds to coplete average Japanese address.

Therefore, I redesigned the interaction so user can immediately start typing the destination using software keyboard.

With the new concept, it took just 10 seconds making it 9 times faster than previous model.

Because the keyboard allows voice input as well, they can enter the address even safer and faster.

After research including internal / external survey, I concluded that 10 key Japanese input method is more common and easier to use for average Mercedes-Benz customers compared to traditional 50 key layout.

So, I redesigned software keybard based on what's already common with smartphones.
